# Классификационный справочник ТН ВЭД ЕАЭС: группы 84 и 87

Практический инструмент для таможенных специалистов и декларантов.
Помогает разграничивать подвижные машины (группа 84) и транспортные
средства (группа 87) при таможенном декларировании.

## Разделы сайта
- **Идентификационные признаки** — сравнительная таблица групп 84 и 87
- **Алгоритм классификации** — интерактивное дерево вопросов (Да/Нет)
- **Судебная практика** — 7 карточек реальных арбитражных дел с фото техники
- **Нормативная база** — ключевые НПА с реквизитами
- **Коды ТН ВЭД** — конкурирующие коды со ставками пошлин

## Технологии
Чистый HTML + CSS + JavaScript, без фреймворков и бэкенда.
